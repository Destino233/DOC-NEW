服务器基本概况​

生电服运行在独立计算机上 有16GB运行内存（分配给服务端12G）

  • 目前服务器运行压力较小，暂不考虑升级内存

服务端：Leaves Github链接(别忘了点个Star) 版本：1.20.1

由于mojang修改了村民交易机制，所以暂时不会升级到新版本
